# HomeRide — ランディングページ

iOS 向け乗換ナビアプリ「HomeRide」の LP。GitHub Pages（`/product-pages/homeride/`）で公開。

## ファイル構成

- `index.html` — 全セクション（注記 → ナビ → ヒーロー → コンセプト → 比較 → 情報設計 → 4画面 → Live Activity → TestFlight フォーム → フッター）
- `styles.css` — 全スタイル。デザインテーマ「OLED ネイティブ」（純黒 × すりガラス、ページ冒頭がロック画面そのもの）
- `script.js` — 演出のみ（スクロールリビール + ナビ境界線）。機能ロジックなし

ビルド工程なし。push したら即公開。**フレームワーク・npm・ビルドの導入は禁止。**

## プロダクト前提

- HomeRide は「スマホをロック解除せずに、ロック画面の Live Activity だけで乗換が完結する」iOS アプリ（iOS 16.1+）
- 解決する課題: 人混みの駅での歩きスマホによる衝突事故を減らす
- ターゲット: 観光客・出張族・営業外回り・休日のお出かけ族 =「不定期利用者」。**通勤者向けの訴求はしない**
- キャッチコピー「ロック画面だけで、乗換が完結。」は**確定・変更不可**

## ⚠ 仕様制約（違反したデザイン・コピー・モックは不可）

1. **番線（ホーム番号）禁止** — 画面モックにもコピーにも一切出さない
2. **相対時間表現禁止** — 「あと◯分」などのカウントダウンは使わない。時刻はすべて「19:50発」のような絶対時刻
3. **登場路線は首都圏の非 JR のみ** — 東京メトロ・都営地下鉄など。JR・山手線・中央線・私鉄は出さない
   - 推奨経路例: 渋谷 →〈半蔵門線〉→ 三越前で乗換 →〈銀座線 浅草方面〉→ 浅草
4. **対応 OS は「iOS 16.1+」** — ウィジェット対応は謳わない（Live Activity と Dynamic Island のみ）
5. **対応路線の注記を残す**: 「東京メトロ全9路線・都営地下鉄・りんかい線・つくばエクスプレス等、首都圏の地下鉄・新交通。JR・大手私鉄は今後検討」（現在 `#testflight` セクション末尾の `.route-note`）
6. **Live Activity モックの情報設計は「乗換後の電車が主役」** — 次の乗換駅・乗換後の路線と行き先・発車時刻を最大サイズ、現在乗車中の情報は最下段に小さく（`.la-card` の構造がこれ。崩さない）

## 技術制約

- 素の HTML/CSS/JS のみ。画像ファイルは極力使わない（全モックは CSS 描画 — 維持すること）
- リンク・アセット参照はすべて**相対パス**（GitHub Pages のサブパス配下で動く必要がある）
- フォントは Google Fonts（現在: Zen Kaku Gothic New 300/400/500/700/900）
- **TestFlight フォームは機能そのまま維持**: Formspree `action="https://formspree.io/f/xaqkoeod"`、フィールドは `nickname` / `email` / `comment` + honeypot `_gotcha`
- ページ冒頭の「本アプリは現在開発中です」注記（`.notice`）を維持
- `prefers-reduced-motion` 配慮を維持（reduced 時は全演出を静的表示）
- **JS 無効でも全コンテンツが読めること**: 演出は `html.js` クラスでゲートしている。`script.js` が `<html>` に `.js` を付けない限り `.reveal` は不透明のまま。この仕組みを壊さない

## デザインシステム（styles.css の `:root`）

- 背景: 純黒 `#000`（OLED）/ ライトセクション `--paper: #ededf2`
- アクセント: `--accent: #ff8a00`（ブランドオレンジ）
- 路線色: `--ginza: #f59a00`（銀座線 G）/ `--hanzomon: #8f76d6`（半蔵門線 Z）/ `--asakusa: #e85298`（都営浅草線 A）
- セクションは黒とライトグレーの交互。すりガラス（`--glass` + backdrop-filter）が共通素材
- モバイルファースト。ブレークポイント: 760px / 860px / 1060px
- フォーム入力の `font-size` は 16px 未満にしない（iOS の自動ズーム防止）
- ヒーローの点灯シーケンス（点灯 → 時計 → Live Activity → コピー）は CSS アニメーションのみで実装
- ヒーロー背景の「夜の駅のアンビエント灯り」（`.hero__ambient` の3層 radial-gradient + `.hero__edge`）は CSS のみ・無画像。暖色＝銀座線オレンジ、寒色＝半蔵門線パープル。`drift` アニメは reduced-motion 時に停止

## 触ってよい / 注意

- コピー微調整・セクション追加は可。ただし上記の仕様制約・確定コピーに抵触しないこと
- フッターの「プライバシー / 利用規約 / お問い合わせ」は現在 `href="#"` のダミー。ページ実装時はここに配線
- 比較セクションの「~12秒 / ~1秒」は操作時間の目安表現（カウントダウンではない）。変更可
