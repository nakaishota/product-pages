## OneTimerについて

昼寝20分、カップ麺3分、筋トレ60秒 — よく使うタイマーをロック画面・コントロールセンターに置いて、ワンタップで開始できるiOSアプリです。

- iOS 26の AlarmKit 採用 — **マナーモード・集中モードでも鳴る**
- Live Activity でロック画面に残り時間を常時表示
- 無料で1スロット / 買い切り¥500で3スロット(サブスクなし)
- App Storeで近日公開予定
